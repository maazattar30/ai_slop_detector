# MiQ AI Content Detector — V1

Classifies YouTube videos into 4 buckets:

| Bucket | Label | Description |
|--------|-------|-------------|
| 🟢 B1 | Human Only | Real camera, natural audio, no AI tools |
| 🔵 B2 | Augmented Human | Human-made, AI used only for editing/post |
| 🟠 B3 | Augmented AI | AI output with human direction/prompting |
| 🔴 B4 | Fully Autogenerated | Text-to-video, deepfake, TTS, AI slop |

## Architecture
12 computed signals (SigLIP, optical flow, audio harmonics, chromatic aberration, compression residual, etc.)
→ Cohen d-weighted module score → Llama 4 Scout (Groq) LLM judge with visual frames
→ 55/45 blended final score + bucket assignment

## Signals
- **SigLIP** zero-shot vision-language AI probability
- **Optical Flow Entropy** — motion chaos (AI = chaotic or static)
- **Chromatic Aberration** — real lenses have R/G/B misalignment
- **Saturation CV** — real scenes have varied colour, AI is uniform
- **Audio Harmonic Ratio** — human speech is highly harmonic
- **Compression Residual** — real frames resist re-compression
- **Spectral Bandwidth** — real audio is spectrally broad
- **Silence Ratio** — AI videos often have no audio
- **Tempo Regularity** — natural audio has timing variation
- **Motion Irregularity** — AI motion is static or uniformly erratic
- **Pitch Variation / Range** — natural speech prosody (conditional on speech detection)

## Setup (HF Space Secrets)
Set these in **Settings → Variables and secrets**:
- `GROQ_API_KEY` — your Groq API key (free tier at console.groq.com)
- `ADMIN_PASSWORD_HASH` — SHA-256 hash of your chosen password

To generate the hash:
```python
import hashlib
print(hashlib.sha256("your_password_here".encode()).hexdigest())
```
